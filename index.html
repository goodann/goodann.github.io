자기소개

중학생 때 처음 Visual Basic으로 개발을 접하고, 고등학교에서 C언어 및 프로그래밍의 기초를 배움
대학에 진학하여 C++를 주로 사용하며 JAVA / Python / C# 을 사용하여 과제 프로젝트 등을 진행 해 본 경험이 있다.
취미로는 주로 컴퓨터를 이용하여 게임, 음악이나 동영상 감상 등을 하며, 운동을 위해 걷기, 자전거 타기를 하기도 한다.
엉뚱하거나 쓸대없는 생각을 잘 한다는 말을 자주 듣지만 이것이야 말로 창의적의 밑거름이라고 생각한다.
병역은 산업기능요원으로 복무를 마쳤음.

프로젝트

테일즈위버 딜레이 계산기 + 메모리 최적화 및 타이머 프로그램

사용기술 : C++, STL, MFC

테일즈 위버 게임을 하는 중, 일정메모리량 이상의 메모리를 테일즈 위버가 사용할 경우 테일즈위버가 종료되는 현상이 커뮤니티를 중심으로 발견되어 이를 해결하려고 주기적으로 테일즈위버의 메모리 사용량을 줄이는 프로그램을 제작, 그 프로그램에 게임에 필요한 딜레이 계산, 보스 재입장 타이머 기능 등 부가기능을 추가하는 프로그램을 제작

마인크래프트 bukkit 서버 관리 프로그램

사용기술 : C++, MFC

마인크래프트 Bukkit을 이용하여 마인크래프트 서버를 열 때, 명령어 입력, 서버 자동 재시작, 서버가 이용할 메모리 설정 로그 저장 등 몇몇 설정을 편하게 하기 위해 제작한 프로그램, bukkit프로세스를 실행하여 그 프로그램의 stdin/stdout을 MFC를 이용하여 GUI상에서 입/출력이 가능하도록 리다이렉션 함

Unity를 이용한 레이싱 게임

사용기술 : C#, Unity
역할 : 아이디어 기획, 플레이어(자동차)움직임 제작, 파티클 이펙트 제작

졸업 작품으로 멀티플레이를 지원하는 물리엔진을 이용한 게임 제작을 진행함, 팀 인원은 4명으로, Unity5를 이용한 레이싱 게임 제작함, 플레이어 자동차의, 부분 파괴를 구현하여 달리면서 실시간으로 플레이어 및 상대의 부위가 파괴는 것을 실시간으로 볼 수 있었다.

Kinect 4대를 이용한 현실 물체를 3D공간상에 복원

사용기술 : C++, OpenCV, OpenGL, libfreenect

Kinect 4대에서 얻은 이미지를 이용하여 3D공간상에 복원하는 기존 시스템을 이용하여, 실시간으로 Kinect에서 데이터를 얻어와 3D공간 상으로 복원시켜 OpenGL을 이용하여 실시간 렌더링 하는 프로그램을 제작.이후 오큘러스 리프트에 적용 가능하도록 확장

스테레오 카메라를 이용한 3D Face Capture

사용기술 : C++, Qt5, OpenGL, OpenCV, nodeeditor

프롬프트 기반의 여러 프로그램으로 나뉘어진 3D얼굴 복원 시스템을 NodeBase의 GUI프로그램으로 이식
연속된 시퀀스 이미지의 Point Tracking GUI프로그램 개발
Wrap3D에서 사용하는 Point데이터를 여러 시퀀스 이미지에서 Point Tracking에서 사용할 수 있도록 2D-3D간 Point Projection하는 프로그램 개발

홀로그램 제작을 위한 데이터 정제 및 씬 세팅을 위한 Unity3D 플러그인 제작

사용기술: C++, C#, Unity

Maya에서 만든 Mesh를 유니티 씬에 원하는데로 입력되도록 제작(좌표계 변환)
Point Cloud 데이터를 유니티 씬에 입력되도록 제작(버텍스 데이터만 존재하는 데이터를 입력되도록 제작)
세팅된 씬 데이터를 홀로그램 제작을 위한 input Data(XML)로 정제하여 출력하도록 플러그인 작성
